Product Design UX | UI

UX means User Experience. There are phases to the UX process and it involves research, understanding the mvp, personas and translating all that information in an organized way using wireframes, information architecture, low and high fidelity mockups, user testing and so much more.

UI means User Interface. This is the front-end phase where visual designs from a UX designer are translated into a technical implementation of the design, ready for developer hand-off.
