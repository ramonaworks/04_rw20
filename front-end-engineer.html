Front End Development and Performance Engineering

Front-end development involves taking a new or existing comp and developing it into an actual web or mobile application or website through the use of HTML, CSS and JavaScript. A lot can be accomplished with vanilla javascript, however, you can use a framework like Angular or a library like React to build out the user interface. Modern JavaScript libraries, tooling. including Node.js, make it possible to truly enjoy the JavaScript ecosystem and push the boundaries of whats possible within a browser.
